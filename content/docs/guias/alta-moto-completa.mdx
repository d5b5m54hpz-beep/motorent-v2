# Alta Completa de una Moto

## Paso 1: Importación
1. Registrar datos de importación (despacho, valores FOB/CIF)
2. Subir documentos de importación

## Paso 2: Registro Inicial
1. Crear moto con datos básicos (marca, modelo, año, color, cilindrada)
2. Cargar datos de identificación (VIN, motor, cuadro)
3. Estado inicial: FUERA_DE_SERVICIO

## Paso 3: Patentamiento
1. Ir a Flota → Patentamiento
2. Seguir checklist RUNA paso a paso
3. Pagar derechos DNRPA (0.8% del valor)
4. Obtener dominio y cédulas
5. Actualizar datos registrales de la moto

## Paso 4: Seguro
1. Contratar póliza
2. Cargar datos de seguro en tab Financiero
3. Subir póliza en tab Documentos

## Paso 5: Disponible
1. Cambiar estado a DISPONIBLE
2. La moto ya está lista para alquilar
